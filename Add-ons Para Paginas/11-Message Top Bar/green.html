jQuery Message Bar

by DPereyra

Introduction

The jQuery Message Bar is a professional and elegant plugin that adds a message bar to your pages.



Usage

Simply reference the relevant files at the head of your page like so:

    <!-- Styles -->
    
    <link rel="stylesheet" href="css/dpMessageBar.css" />
    
    <!-- jQuery -->
    
    <script type="text/javascript" src="js/jquery-1.5.2.min.js"></script>
    
    <!-- required plugins -->
    
    <script type="text/javascript" src="js/jquery.dpMessageBar.js"></script>
Initialitation

    $(document).ready(function(){
       $("#dpMessageBar").dpMessageBar();
    });

#dpMessageBar is the ID of the UL HTML Object that will use this plugin.

    <ul id="dpMessageBar">
        <li>Message 1...</li>
        <li>Message 2...</li>
        <li>Message 3...</li>
        <li>Message 4...</li>
    </ul>



Options

  • loop

    Signature:  $("#dpMessageBar ").dpMessageBar ({loop: true })
    Set whether to loop is active.
    Default value is true
  • autoload
    • Signature:  $("#dpMessageBar ").dpMessageBar ({autoload: true })
    Set if showing the message bar when the page loads.
    Default value is true
  • speedAnimation
    • Signature:  $("#dpMessageBar ").dpMessageBar ({speedAnimation: 500 })
    Set the speed animation in miliseconds (300, 500, 1000, 2000, etc) or "slow" and "fast".
    Default value is "slow"
  • delay
    • Signature:  $("#dpMessageBar ").dpMessageBar ({delay: 1000 })
    Set the delay in miliseconds (300, 500, 1000, 2000, 5000, etc).
    Default value is 3000
  • theme
    • Signature:  $("#dpMessageBar ").dpMessageBar ({theme: "navy" })
    Set the color theme. Possible values are: Cream, Navy, Green, Red, Chocolate
    Default value is "cream"
  • showArrows
    • Signature:  $("#dpMessageBar ").dpMessageBar ({showArrows: false })
    Set if showing the arrows or not.
    Default value is "true"
  • position
    • Signature:  $("#dpMessageBar ").dpMessageBar ({position: "top" })
    Set the position of the bar. Possible values are: top, fixed
    Default value is "fixed"



Using Links

You can use links on the messages normally:

    <ul id="dpMessageBar">
        <li>Message 1... click <a href="#">here</a></li>
        <li>Message 2... click <a href="#">here</a></li>
